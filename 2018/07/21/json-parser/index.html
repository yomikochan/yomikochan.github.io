此次更新的主要内容

  1. 增加了通过key获取/设置对应的value的api:getvalue()、setvalue()；
  2. 增加了通过key删除key-value的api；

  4. 重载了[]符，使其能够通过字符串key来获取或设置value值。但并不完善。只能对一种类型的value进行操作；
  5. 增加了许多内置转换函数：bits()、float2str()、float2ch_arr()、int2str()、con_char2int；
